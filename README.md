# lovli.js
A starter kit which allows you to develop realtime applications in minutes.

Utilizing React, Redux, Horizon and webpack.

### Installation
```
git clone git@github.com:flipace/lovli.js.git
cd lovli.js
npm i
```

### Run
```
npm start
```
